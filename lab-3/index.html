ITC505 Lab 3: Steps to Customise a Font

Font-Name: Henny Penny

To use a custom font like Henny Penny on your website, you first need to select a unique font that isn't commonly found on users' systems. Start by accessing Google Fonts to find and download the desired Henny Penny font variants. After selecting the variants you need, download the font files to your computer.

Next, upload the downloaded font files to the assets directory on your web server. This directory will store all your custom font files. Then, create a CSS styling file named styles.css, which will contain the custom styles for your website. Within this CSS file, define a @font-face rule that specifies the font-family name and the path to the font files in the assets directory.

Finally, link the styles.css file to your HTML document, typically index.html, by adding a link tag in the head section. This will ensure that the custom styles are applied across your website. Once you've updated and uploaded the HTML and CSS files to the website directory, the Henny Penny font should be displayed across your website, providing a unique and consistent appearance.
